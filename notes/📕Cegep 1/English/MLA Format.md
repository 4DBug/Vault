# MLA Format

- Double spaced
	- Use 2x spacing instead of actually inserting newlines
- 12 pt
- Bibliography reverse indented

```html
<author>
<instructor>
<course>
<date>

							<title>

<content>

						  Works Cited

<bibliography>
```
